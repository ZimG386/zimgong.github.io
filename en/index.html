Wikipedia Search Engine

Aug 2023 - Present

Github Link

A full search engine using vector space model techniques. It uses 1M wikipedia entries as demo content.

The engine uses tokenizer and indexing to store information, and accepts different ranking models including BM25, Dirichlet LM, etc. It supports Document Augmentation with trained LLM models. The user can also choose to use machine learning models or cross-encoder ranking as rankers. Finally, MAP and NDCG metrics are provided to evaluate the effectiveness.

Pi-Billiard Assistant

Jan 2023 - May 2023

Github Link

The Pi Billiard Assistant is an embedded system that consists of a Raspberry Pi 4, a Raspberry camera and an external laptop providing a dynamic trajectory estimation based on player"s cue direction. The system is capable of calculating the trajectory with a collision between cue ball and cushions. Players can better adjust the shot by referring to the trajectory and explore more potential solutions to snooker from laptop display. The system is installed with an external frame for video capturing and leaves space for future stretch designs and implementations.

Sep 2022 - Dec 2022

Github Link

This is a project that aims to compare multiple classification methods including KNN and CNN network in Alzheimer’s disease diagnosis, reach higher accuracy with only single-angled MRI dataset and use GAN network to reconstruct brain MRI photos of Alzheimer’s disease patients, which are possible to then be fed back to the classification network to test its result.
